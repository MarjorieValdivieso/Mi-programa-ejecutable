
# POLIPERROS

Este proyecto es un programa en Python que permite **registrar, mostrar y buscar Poliperros**, almacenando información básica como nombre, huella dactilar.


## Instalacion

Para que mi proyecto sea ejecutable utilice 

```bash
  pip install pyinstaller
```
    

## Contenido del proyecto

- `archivos.py` — Archivo principal del programa.  
 
- `poliperros.txt` — Archivo donde se guarda la base de datos de los Poliperros.
